Grace Murray Hopper

Grace Murray Hopper

Theory of machine-independent programming languages, and the FLOW-MATIC programming language

Studies:

Mathematics and mathematical physics

Fields

  • Computer science
  • Mathematics

Major Milestones

  • Part of the team that developed the UNIVAC I computer
  • One of the first programmers of the Harvard Mark I computer
  • Pioneer of computer programming
  • First to devise the theory of machine-independent programming languages
  • Creator of the FLOW-MATIC programming language used to create COBOL

Frances Elizabeth Allen

Frances Elizabeth Allen

Invento

Studies:

Mathematics

Fields:

  • Computer science
  • High-performance computing
  • Parallel computing
  • Compilers
  • Optimizing compilers

Major Milestones

  • Professor at IBM Research in Poughkeepsie, New York
  • Assigned to the Harvest project for code breaking with the National Security Agency, and worked on a programming language called Alpha
  • Managed the compiler-optimization team for both Harvest and the Stretch project
  • She contributed to the ACS-1 project at Thomas J. Watson Research Center helping to improve the efficiency of machine code translated from high-level languages
  • Led IBM's work in the developing parallel computing area, and helped to develop software for the IBM Blue Gene project
  • After retiring, she remained active in programs that encourage women and girls to seek careers in science and computing
